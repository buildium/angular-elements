bdPopover

  1. - directive in module buildium.angular-elements.popover

Attaches a popover tooltip to an element with custom content.

Directive Info

  • This directive executes at priority level 0.

Usage

  • as element: 
    <bd-popover
  selected="[object Object]"
  link-id="[object Object]"
  pointer="[object Object]"
  popover-container-class="[object Object]"
  popover-link-class="[object Object]"
  pointer-class="[object Object]"
  show-on-hover="[object Object]"
  on-link-click="[object Object]">
...
</bd-popover>

Arguments

Param Type Details
selected boolean
linkId string
pointer boolean
popoverContainerClass string
popoverLinkClass string
pointerClass string
showOnHover boolean
onLinkClick function

Examples

<bd-popover selected="false" 
            link-id="Ygritte" 
            pointer="true" 
            show-on-hover="true" 
            link-class="popover-link" 
            container-class="'popover'" 
            pointer-class="'popover__pointer'" 
            show-on-hover="true" 
            on-link-click="controller.onClick()">
    <popover-link>
        <span class="tooltip">Ygritte quotes</span>
    </popover-link>
    <popover-body>
        <h1>You know nothing, Jon Snow</h1>
    </popover-body>
</bd-popover>
