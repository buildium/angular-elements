BdSubmenu

  1. - service in module buildium.angular-elements.popover

Control the positioning and display of the popover element

Dependencies

  • $rootScope
  • $timeout
  • $document

Usage

BdSubmenu();

Methods

  • display(elem);

    Positions popover against the given element (elem) and displays it on the page

    Parameters

    Param Type Details
    elem DOMElement

  • startTimer();

    Start the timer if there is an element that the BdSubmenu service considers open. At the end of the time, the popover is closed. This ignores the mouseLeave event that may be fired on a submenu as it is in the process of fading out.

  • stopTimer();

    Cancels any timer started by startTimer

  • closeAll();

    Closes any active popover elements

  • positionPopoverBody(elem);

    Positions popover against the given element (elem)

    Parameters

    Param Type Details
    elem DOMElement
