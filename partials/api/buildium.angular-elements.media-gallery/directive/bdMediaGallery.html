bdMediaGallery

  1. - directive in module buildium.angular-elements.media-gallery

Displays a gallery of media items (images, etc...)

Directive Info

  • This directive executes at priority level 0.

Usage

  • as element: 
    <bd-media-gallery
  media=""
  allow-remove="Boolean"
  on-remove=""
  on-select="">
...
</bd-media-gallery>

Arguments

Param Type Details
media object[]

media elements to display in the gallery. Expected keys are:

  • string fileName the name of the file, recognized by the user
  • string title a title to display for this media as alt text
  • string imageUrl the url for an image to display for this media
  • boolean isRemovable=true whether to show option to remove this item from gallery, takes precedence over allowRemove
allowRemove
(optional)
 Boolean

whether to show option to remove on items in gallery

(default: true)
onRemove
(optional)
 Function

callback to be executed when user removes an item
onSelect
(optional)
 Function

callback to be executed when user selects an item

Examples

                {
                    fileName: 'woman-blowing-pink-powder.jpeg',

                    imageUrl: 'https://images.pexels.com/photos/612977/pexels-photo-612977.jpeg?h=350&dpr=2&auto=compress&cs=tinysrgb'
