bdMediaGallery

  1. - component in module buildium.angular-elements.media-gallery

Displays a gallery of media items (images, etc...)

Usage

  • as element: 
    <bd-media-gallery
  media="object[]"
  [allow-remove="Boolean"]
  [on-remove="Function"]
  [on-select="Function"]
  [allow-enlarge="Boolean"]
  [on-enlarge="Function"]
  [limit-to="Number"]
  [on-view-all="Function"]
  [enable-list-view="Boolean"]
  [on-set-view="Function"]>
...
</bd-media-gallery>

Arguments

Param Type Details
media object[]

media elements to display in the gallery. Expected keys are:

  • string fileName the name of the file, recognized by the user
  • string title a title to display for this media as alt text
  • string imageUrl the url for an image to display for this media
  • boolean isRemovable=true whether to show option to remove this item from gallery, takes precedence over allowRemove
allowRemove
(optional)
 Boolean

whether to show option to remove on items in gallery

(default: true)
onRemove
(optional)
 Function

callback to be executed when user removes an item
onSelect
(optional)
 Function

callback to be executed when user selects an item
allowEnlarge
(optional)
 Boolean

whether to show prompt to enlarge image

(default: false)
onEnlarge
(optional)
 Function

callback to be executed when user clicks enlarge prompt
limitTo
(optional)
 Number

limits the number of items visible in the gallery
onViewAll
(optional)
 Function

if limitTo provided, callback to be executed when user chooses to view all items
enableListView
(optional)
 Boolean

whether to show option to toggle between list view and tile view

(default: false)
onSetView
(optional)
 Function

if listView provided, callback to be executed when user chooses to display list view
