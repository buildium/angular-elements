bdNavigation

  1. - directive in module buildium.angular-elements.navigation

A flexible component for building all types of navigation elements. CSS classes are generously added throughout to enable style overrides for horizontal, tabbed, or vertical navigation bars.

Directive Info

  • This directive executes at priority level 0.

Usage

  • as element: 
    <bd-navigation
  nav-items="Array">
...
</bd-navigation>

Arguments

Param Type Details
navItems
(optional)
 Array

The collection of navigation items that should be added

  • string title - Name to display for this navigation item
  • boolean isEnabled - Whether user is able to interact with this navigation item
  • boolean isActive - Whether this navigation item is currrently active (i.e. we are on that page)
  • string href - Location that this navigation item points to
