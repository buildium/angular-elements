bdVerticalNavigation

  1. - directive in module buildium.angular-elements.navigation

A vertical navigation element suitable for sidebar navs

Directive Info

  • This directive executes at priority level 0.

Usage

  • as element: 
    <bd-vertical-navigation
  navigation-items="Array">
...
</bd-vertical-navigation>

Arguments

Param Type Details
navigationItems
(optional)
 Array

The collection of navigation items that should be added

  • string title - Name to display for this navigation item
  • boolean isEnabled - Whether user is able to interact with this navigation item
  • boolean isActive - Whether this navigation item is currrently active (i.e. we are on that page)
  • string href - Location that this navigation item points to
  • string cssClass - Custom class(es) to add to this navigation item
  • array menu - a sub-navigation with the same collection structure as navigationItems

Examples

<div class="page-content" ng-controller="ExampleController as vm">
    <bd-vertical-navigation navigation-items="vm.navItems" class="navigation--large"></bd-vertical-navigation>
</div>
angular.module('exampleModule', ['buildium.angular-elements.navigation'])
.controller('ExampleController', function () {
    this.navItems = [
        {
            title: 'Appearance',
            isEnabled: true,
            isActive: false,
            href: 'https://appearance.example.com'
        },
        {
            title: 'Policies',
            isActive: true,
            href: 'https://policies.example.com'
        },
        {
            title: 'Manage',
            href: 'https://manage.example.com'
        }
    ]
})
