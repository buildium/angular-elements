bdAccordionToggle

  1. - directive in module buildium.angular-elements.accordion

Element responsible for toggling a bdAccordionGroup open or closed

Directive Info

  • This directive executes at priority level 0.

Usage

  • as attribute: 
    <ANY
  is-open="Boolean"
  disabled="Boolean"
  on-change=""
  open-class="String"
  disabled-class="String"
  toggle-class-name="String">
...
</ANY>

Arguments

Param Type Details
isOpen
(optional)
 Boolean

is toggle open by default
disabled
(optional)
 Boolean

is toggle disabled
onChange
(optional)
 Function

change event to fire when open state changes
openClass
(optional)
 String

class to add when toggle is open
disabledClass
(optional)
 String

class to add when toggle is disabled
toggleClassName
(optional)
 String

changes toggle trigger to element matching this class name

Examples

<style>
    .hide { display: none; }
</style>
<section class="page-content">
    <bd-accordion>
        <ul>
            <li bd-accordion-toggle is-open="false">
                This accordion group starts closed
                <ul bd-accordion-group>
                    <li class="list-item list-item--bulleted">Hello world</li>
                    <li class="list-item list-item--bulleted">Foo Bar</li>
                </ul>
            </li>
            <li bd-accordion-toggle is-open="true">
                This accordion group starts open
                <ul bd-accordion-group>
                    <li class="list-item list-item--bulleted">Hello world</li>
                    <li class="list-item list-item--bulleted">Foo Bar</li>
                </ul>
            </li>
            <li bd-accordion-toggle is-open="false">
                This accordion group starts closed
                <ul bd-accordion-group>
                    <li class="list-item list-item--bulleted">Hello world</li>
                    <li class="list-item list-item--bulleted">Foo Bar</li>
                </ul>
            </li>
        </ul>
    </bd-accordion>
</section>
